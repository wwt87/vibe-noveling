# 返修策略池

## SSoT 运算规则

### 基本操作：Sum-Mod

1. 生成随机字符串（至少 20 字符，混合大小写字母和数字）
2. 将指定片段中每个字符的 ASCII 码值相加
3. 用 ASCII 和对候选项数量取模，得到选项索引

```
例：字符串片段 "kR7pL2m"
ASCII 值：k=107, R=82, 7=55, p=112, L=76, 2=50, m=109
总和 = 591
如果有 3 个候选策略 [动作展开, 人物反应层, 感官与内心]：
591 mod 3 = 0 → 选中"动作展开"
```

### 分段复用

一个随机字符串分 3 段，每段驱动一个候选版本。每段内部再做微决策：

```
随机字符串: "kR7pL2mN4bT9xQwJ5aHc"
├── "kR7pL2m" → 候选 A
│   ├── 策略选择：Sum-Mod(全段) → 从策略池选一种
│   ├── 句式选择：Sum-Mod(前3字符) → 从句式池选一种
│   ├── 感官选择：Sum-Mod(中3字符) → 从感官池选一种
│   └── 词汇选择：Sum-Mod(后1字符) → 从词汇池选一种
├── "N4bT9xQ" → 候选 B（同上结构）
└── "J5aHc"   → 候选 C（同上结构）
```

### 操作规则

- 每个标记必须生成独立的随机字符串，不跨标记复用
- 3 个候选必须分别走不同策略，不允许两个候选走同一种策略
- 计算过程在 thinking 中完成
- 必须忠实执行 SSoT 的结果，不得在计算后因"觉得不合适"而换一个
- 同一章同类型标记的多个实例之间，SSoT 自然保证策略分配不同（不同字符串→不同取模结果）

## 扩写策略池（加粗标记）

将一句话展开为多个连贯长句，像作者写兴奋了多写几笔。

### 策略 A：动作展开

核心手法：拆复合动作为时间序列，加具体化操作步骤。

- 把"同时发生"的动作拆成时间序列，每个动作独立成句
- 技术动作后加解释性补丁——"用XX（方式）""这是一道XX（命名）"
- SSoT 驱动：拆成几步、用什么衔接、是否加命名

进阶操作：

1. **拆复合动作为序列**：把"同时发生"的动作拆成时间序列，每个动作独立成短句
   - 原文：不需要停顿，不需要蓄力，三道火红阵纹从粉末中亮起来。
   - 改写：与此同时，左手拇指滑过通灵玉佩，一道光影投射在粉末上。三道火红阵纹瞬间从地面亮起来。火灵阵，成了！

2. **加具体化操作步骤**：技术动作后加解释性补丁——"用XX（方式）""这是一道XX（命名）"
   - 原文：催动灵力，掌心一送，火球破空而出。
   - 改写：催动灵力，掌心一送，一个火球破空而出。这是一道火球术。

### 策略 B：人物反应层

核心手法：加旁观者反应，插入对话交锋，增加现场张力。

- 关键动作后加旁观者反应——一句话、一个群体行为
- 对峙场景插入简短对话回合，每句不超过十个字
- SSoT 驱动：选什么反应、对话几句、谁先开口

进阶操作：

1. **加旁观者反应**：关键动作后加一句旁观者的具体反应——一句话（"好快！"）、一个群体行为（叽叽喳喳却不敢上前）
   - 原文：火灵阵，成了。围观的人一下子安静了。
   - 改写：火灵阵，成了！"好快！"围观的人一下子安静了。

2. **对峙场景加对话回合**：插入简短的对话交锋，每句不超过十个字，用来打节拍
   - 原文：苏寒染站在那里，嘴角咧开。
   - 改写："逃？""怎么，打算投降么？"苏寒染摇摇头："你要不先看看情况？"

### 策略 C：感官与内心

核心手法：加感官铺陈，加口语化内心独白，增加沉浸感。

- 展开环境感官细节（光影/气味/温度/声响/质感）
- 插入口语化内心独白，可以带脏话、疑问、语气词
- SSoT 驱动：偏重哪种感官、独白的情绪基调、长短

进阶操作：

1. **加感官铺陈**：展开环境细节，选一种感官通道深入
   - 原文：他走进了洞府。
   - 改写：洞府里阴凉得很，空气里有一股陈年的霉味，脚底踩到的东西沙沙响。

2. **加口语化内心独白**：把旁白式心理分析改成角色内心的口语独白，可以带脏话、疑问、语气词
   - 原文：炼气五层以上，硬来不行。
   - 改写：妈的，这怎么打，赶紧溜溜球。

## 简写策略池（删除线标记）

将标记内容压缩为更紧凑的表达。

### 策略 A：合并压缩

核心手法：多短句合并为一个通顺长句。

- 重新改写整个句子，整合语义、调整语序、添加连接词
- 不能只是简单加逗号串联
- 适当加入"在…下""连…都""时"等结构词
- SSoT 驱动：合并后的句式结构、连接词选择

### 策略 B：概括替代

核心手法：用一个精准的动作/判断替代多句铺陈。

- 旁白描述转角色判断："跟XX一样" → 角色视角的直接判断
- 用直接形容词替代文学比喻
- 用时间对比替代镜头感描写
- SSoT 驱动：从什么视角概括、保留哪个信息点

进阶操作：

1. **旁白描述转角色判断**：把"XX像/如同/跟XX一样"的旁白比喻，改成角色视角的直接判断
   - 原文：跟路边普通鹅卵石没什么两样。
   - 改写：苏寒染掂了掂，就是块普通石头。

2. **用时间对比替代镜头感**：把画面式变化描写改成时间锚点对比
   - 原文：粉袋肉眼可见地瘪下去。
   - 改写：早上粉袋还鼓鼓囊囊的，一下午过去，去了大半。

3. **用直接形容词替代文学比喻**：简化过度修饰
   - 原文：唯独眼神不普通，像在掂量从哪儿下刀最快。
   - 改写：眼神颇为阴厉。

### 策略 C：留白裁剪

核心手法：删掉修饰和过程，只留关键动作和结果。

- 删掉后不影响主角推进的直接删除
- 删掉过度解释的部分，让读者自己感受
- 只保留动作 + 结果 + 情绪暗示
- SSoT 驱动：保留哪些关键信息、如何暗示情绪

## 润色策略池（斜体标记）

调整句子使其通顺、有节奏、不卡。吸收了原"润滑"和"合并"两种操作。

### 策略 A：补过渡

核心手法：加感知步骤、因果连接词、自然过渡，让句子顺滑。

- 加感知步骤：角色获取信息时，加"怎么发现的"动作，把信息获取拆成两步
- 加因果连接词和限定词：句间加"于是""所以""这""又"
- 远景区加"可见"框架：暗示角色视角而非全知信息
- SSoT 驱动：补哪种过渡、加在哪个位置

进阶操作：

1. **加感知步骤**：角色获取信息时，加一个"怎么发现的"动作
   - 原文：屏幕上弹出异常警示——通讯阵灵力波动偏移。
   - 改写：屏幕上弹出异常警示：通讯阵灵力波动有偏移。苏寒染打开看了一眼，是白天布的通讯阵。

2. **加因果连接词和限定词**：在独立短句末尾加补充短语，句间加连接词
   - 原文：三团对外联络的命脉全系在这阵上。翻身起来
   - 改写：三团对外联络的命脉全系在这阵上，不能出事。于是又翻身起来

3. **远景区加"可见"框架**：远景描写前加"可以看到""能看见"
   - 原文：远处营地轮廓浮出来，帐篷塌了大半
   - 改写：远处营地轮廓浮现出来，可以看到帐篷塌了一大半

### 策略 B：句式重构

核心手法：调整语序、改变句式结构。

- 重新安排信息出现的顺序（先果后因/先因后果/先动作后感受）
- 加入结构词："在…下""连…都""时"
- 合并后确保语序自然，不产生歧义
- 不能只是简单加逗号、顿号把短句串联起来
- SSoT 驱动：选哪种句式结构、从什么角度切入

### 策略 C：感官化改写

核心手法：用身体感受和具体细节替代抽象或干硬的描述。

- 把抽象描述改成感官化的具体感受
- 用动作/环境/身体反应间接表达情绪
- 加入质感、温度、声音等具体感知
- SSoT 驱动：偏重哪种感官、保留哪些信息、替换哪些词

## 执行约束

1. **严格限定范围**：只修改被标记的文字及其紧邻上下文（如需补连接词）
2. **不改剧情**：返修不改变事实信息、动作因果和剧情点职责
3. **不改风格**：返修不改变正文的整体文风，只调整被标记的局部
4. **每次只处理一个标记**：处理完写入文件后再处理下一个
5. **同一章同类型标记不趋同**：SSoT 确保同类标记的策略分配不同
